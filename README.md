# Web Technology Laboratory – 22CS68L

## List of Programs

### 1. Department Timetable Web Page
Design a web page for the department timetable containing:
- Suitable header and footer  
- Description of course codes and titles  
- Faculty names and initials  
- Use of:
  - href links (`<a>`)
  - List tags  
  - University logo as background  
  - Table tag with:
    - Row/column spanning  
    - Borders  
- Refer to your class timetable for content

### 9. Form Validation (Advanced)

Validate the following:

- **User ID:** 5–12 characters  
- **Name:** Only alphabets, minimum 15 characters  
- **Password:** At least 8 characters with:
  - One uppercase letter  
  - One special character  
  - Alphanumeric characters  
- **Email:** Valid email format
